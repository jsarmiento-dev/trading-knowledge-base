# Perfil de Volumen (Volume Profile)

**Fuente:** NovaTrader — BITCOIN HOY ¿HASTA DONDE PUEDE SUBIR? (2) (ID: x7O40C-A_U0)

## Descripción
El perfil de volumen muestra la cantidad de volumen negociado en cada nivel de precio durante un período determinado. Nova lo utiliza extensamente para identificar zonas de interés.

## Conceptos clave
- **POC (Point of Control):** el nivel de precio donde se ha negociado el mayor volumen. Es la zona más relevante
- **Nodo de alto volumen (High Volume Node - HVN):** área donde hubo mucha negociación. Actúa como soporte/resistencia
- **Nodo de bajo volumen (Low Volume Node - LVN):** área con poca negociación. El precio tiende a atravesarlas rápidamente (desequilibrio)

## Aplicaciones
- El POC y nodos de alto volumen definen zonas de **entrada y salida**
- Cuando el precio llega a un nodo de alto volumen, suele encontrar rechazo o soporte
- Las zonas de bajo volumen son áreas donde el precio se acelera

## Ejemplos del video
1. **BTC:** nodo de alto volumen en $63,000 → barrido → rechazo al alza
2. **BTC:** POC de rango como zona de posible rechazo (si pierde el nodo superior)
3. **Ethereum:** POC en $1,800 → barrido de largos → divergencia → subida violenta
4. **Ethereum:** nodo ~$2,050 como soporte actual; si lo pierde → continuación a $1,975

## Cómo usarlo
- Tirar el perfil de volumen desde rangos relevantes (ej. desde el LPSI de una distribución)
- Identificar el POC y nodos de alto volumen
- Usarlos como zonas de soporte/resistencia dinámicas
- Combinar con Fibonacci y divergencias RSI para entradas precisas
