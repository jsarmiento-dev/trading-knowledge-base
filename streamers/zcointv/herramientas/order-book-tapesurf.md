## Definición
TapeSurf (anteriormente Okotoki) es una terminal gratuita de visualización de Order Book y Order Flow para criptomonedas. Permite leer en tiempo real dónde se colocan las órdenes límite de compra y venta, complementando el análisis técnico tradicional basado en gráficos históricos.

**Fuente:** ZCoinTV

## Conceptos Clave
- **Order Book (Libro de Órdenes):** Representación en tiempo real de todas las órdenes límite de compra y venta colocadas en un exchange. Es donde realmente ocurre el trading; el gráfico de velas es solo el registro histórico
- **Order Flow:** Estudio del flujo de órdenes que permite visualizar dónde se concentran las órdenes grandes. TapeSurf lo muestra como Heatmap superpuesto al gráfico de precios
- **Cada activo y plataforma tiene su propio libro de órdenes:** No es lo mismo el orderbook de Binance que el de Bitget. Hay que usar el que corresponde al exchange donde se opera
- **TradingView NO tiene Order Book:** Solo muestra gráficos históricos. Para ver el libro de órdenes se necesitan plataformas específicas

## Configuración y Lectura
1. **Plataforma:** TapeSurf (tapesurf.com, antes okotoki.com). Versión gratuita suficiente
2. **Registro:** Crear cuenta con Google para guardar configuraciones personalizadas
3. **Order Book básico:**
   - Línea verde horizontal = precio actual
   - Barras verdes debajo = órdenes de compra (límites de compra por debajo del precio)
   - Barras rojas arriba = órdenes de venta (límites de venta por encima del precio)
   - Número en la barra = cantidad de BTC/cripto en ese nivel
   - Número a la derecha = cantidad acumulada hasta ese nivel (cuánto hay que comprar/vender para llegar ahí)
4. **Order Flow Heatmap:**
   - Fusiona orderbook + gráfico
   - Configurar sensibilidad desde la rueda dentada (valores 10-20 recomendados)
   - Números en el heatmap = cantidad de órdenes en cada punto
   - Zonas con muchas órdenes → precio se desacelera (soporte/resistencia)
   - Zonas con pocas órdenes → precio se acelera (LVNs, zonas de ruptura)

## Reglas Operativas
1. Usar el Order Book como complemento del análisis técnico, NUNCA como única herramienta
2. Validar niveles del Volume Profile contrastándolos con el Order Book: donde hay alto volumen en el perfil debe haber concentración de órdenes
3. Donde hay concentración de órdenes → posible soporte/resistencia → precio puede frenar o rebotar
4. Donde hay vacío de órdenes → precio tiende a acelerarse → posible continuación de tendencia
5. El precio tiende a magnetizarse hacia zonas con alta concentración de órdenes
6. Precaución con el Spoofing en exchanges crypto: hay operadores que colocan órdenes grandes falsas para manipular la percepción del libro. Ha disminuido con la regulación, pero sigue existiendo
7. El Order Book habla del PRESENTE, el gráfico habla del PASADO — combinarlos es la ventaja clave

## Frase Destacada
> "El trading no sucede en los gráficos. El gráfico es simplemente una representación histórica de lo que ha pasado en el libro de órdenes."
